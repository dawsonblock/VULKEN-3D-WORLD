# Quickstart

This guide shows how to export voxel data generated by VULKEN into a
standalone HTML file using `scripts/export_world_to_html.py`.

## Installation

Install Python dependencies:

```sh
pip install -r requirements.txt
```

## Exporting a world

Create or load a voxel array and write it to ``.npy`` format:

```sh
python - <<'PY'
import numpy as np
voxels = np.zeros((2, 2, 2), dtype=np.uint8)
voxels[0, 0, 0] = 1
np.save('voxels.npy', voxels)
PY
```

Convert the array into a minimal `three.js` scene:

```sh
python scripts/export_world_to_html.py voxels.npy world.html
```

Open `world.html` in a web browser. You should see a single cube rendered in
the scene, proving the export succeeded.

An excerpt from the generated HTML looks like this:

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>VULKEN Voxel Export</title>
    <script src="https://cdn.jsdelivr.net/npm/three@0.160/build/three.min.js"></script>
  </head>
  <!-- ... -->
</html>
```
